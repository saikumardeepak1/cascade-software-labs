# Cascade Software Labs

Enterprise Agentic AI Solutions — Marketing website built with Next.js 16, React 19, Tailwind CSS v4, Framer Motion, and GSAP.

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **UI**: React 19, Tailwind CSS v4, shadcn/ui
- **Animation**: Framer Motion, GSAP + ScrollTrigger, Lenis smooth scroll
- **3D**: Spline (@splinetool/react-spline)
- **Font**: Satoshi (local woff2)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Pages

- `/` — Home
- `/solutions` — Service Offerings
- `/company` — About Us
- `/careers` — Join the Team
- `/projects` — Projects & Products
- `/contact` — Get in Touch

## Deployment

Deployed on Vercel. Connect your GitHub repo to Vercel for automatic deployments.

```
vercel deploy
```
